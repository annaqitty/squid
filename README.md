# Squid Proxy V.4.1

Requirements

  - Ubuntu (14 > 20 LTS)
  - SSH Access
  - Port is not used


### Installation

Install cURL [IMPORTANT]

```sh
$ sudo apt-get install curl -y
```

Then run the installer command
```sh
$ sudo curl -sL https://raw.githubusercontent.com/annaqitty/squid/master/squid_ubuntu.sh -o anna.sh && sudo bash anna.sh
```
